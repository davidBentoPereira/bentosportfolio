# Bento's Portfolio :seedling:

This is 

## Stack

- Ruby 2.7.0 :gem:
- Ruby on Rails 6.0 :gem:

## Installation

1 -  Start by cloning the project :

```
git clone url_of_the_project
```

2 - Place yourself into the project's folder :

```
cd bentosportfolio
```

3 - Install the gems :gem: :

```
bundle install
```

4 - Done :tada: !
